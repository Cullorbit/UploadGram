UploadGram — это Android‑приложение, предназначенное для автоматической загрузки фотографий и видео из указанных папок вашего устройства напрямую в чат Telegram. Это удобный инструмент для создания резервных копий ваших медиафайлов или для их быстрой отправки в групповой чат или канал.

✨ Возможности
-------------

- **Выбор нескольких папок:** Добавляйте и настраивайте неограниченное количество папок для синхронизации.
- **Интеграция с Telegram:**
  - Используйте собственного Telegram‑бота для загрузки файлов.
  - Поддержка отправки в темы форумов (супергрупп).
- **Фоновая работа:** Приложение работает в фоновом режиме, сканируя папки и загружая новые файлы без вашего участия.
- **Гибкие настройки:**
  - Задайте глобальные настройки для Telegram (токен бота).
  - Установите индивидуальные параметры для каждой папки.
  - Настройте интервал синхронизации.
  - Выберите тип загружаемых файлов: только фото, только видео, только аудио или всё вместе.
  - Включите опцию синхронизации только при подключении к Wi‑Fi для экономии мобильного трафика.
- **Удобный интерфейс:**
  - Простой и понятный интерфейс со списком папок.
  - Встроенный лог для отслеживания статуса загрузки файлов.
  - Поддержка светлой и тёмной тем оформления.

🛠️ Сборка из исходного кода
----------------------------

1. Клонируйте репозиторий:  
   `git clone https://github.com/ваш-репозиторий/PhotoUploaderApp.git`
2. Откройте проект в Android Studio.
3. Соберите проект, нажав на `Build` -> `Make Project`.
4. Запустите приложение на эмуляторе или реальном устройстве.
