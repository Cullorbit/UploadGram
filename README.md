UploadGram is an Android application designed for automatically uploading photos and videos from selected folders on your device directly to a Telegram chat. It is a convenient tool for creating backups of your media files or quickly sending them to a group chat or channel.

✨ Features
----------

- **Multiple folder selection:** Add and configure any number of folders for synchronization.
- **Telegram integration:**
  - Use your own Telegram bot to upload files.
  - Support for sending to forum topics (supergroups).
- **Background operation:** The app runs in the background, scanning folders and uploading new files without your intervention.
- **Flexible settings:**
  - Set global Telegram settings (bot token).
  - Configure individual parameters for each folder.
  - Adjust the synchronization interval.
  - Choose file types to upload: photos only, videos only, audio only or both.
  - Enable sync only on Wi‑Fi to save mobile data.
- **User-friendly interface:**
  - Simple, clear interface with a list of folders.
  - Built-in log to track upload status.
  - Support for light and dark themes.

🛠️ Building from source
------------------------

1. Clone the repository:  
   `git clone https://github.com/ваш-репозиторий/PhotoUploaderApp.git`
2. Open the project in Android Studio.
3. Build the project via `Build` -> `Make Project`.
4. Run the app on an emulator or a real device.
